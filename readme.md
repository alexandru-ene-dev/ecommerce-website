# E-commerce Website

A fullstack E-commerce Website built with Typescript, React, Express and Mongoose

---

## Tech Stack

- **Frontend**: React + TypeScript + CSS
- **Backend**: NodeJS + Express + TypeScript
- **Database**: MongoDB + Mongoose
- **Authentication**: JWT (JSON Web Token)

---

## Features

- **User Authentication**: Secure registration, login, and logout functionality
- **Theme Support**: Toggle between light and dark modes for personalized user experience
- **Profile Management**: Update avatar, username, and password in the profile settings
- **Product Catalog**: Browse available products, add them to favorites or the shopping cart
- **Search & Filtering**: Quickly find products with search and filter options
